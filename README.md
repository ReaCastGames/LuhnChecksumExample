This is a simple project that was built to run in Intellij IDEA 2026.1.1, utilizing its console for input/output.

The Checksum class runs the Luhn algorithm as described in the Luhn Algorithm wikipedia page, using the Mod 10 formula for the check digit.

The ChecksumUnitTests class runs junit tests on the Checksum class with 100% code coverage and 100% passing rate on expected outcomes.

This project does not cover the limitations of the Luhn algorithm, such as two-digit transpositions or twin errors.
